# Bloom Scroller

An infinite scroll webapp showcasing real nature observations from iNaturalist.
Built with React + Python + Render + Docker

## Features
- Infinite scroll with iNaturalist API observations
- Filter by taxa (Plants, Fungi, Animals, Insects)
- Minimal, responsive design
- Fast image loading with lazy loading

## Tech Stack
- Frontend: React + Vite
- Backend: Python
- External: iNaturalist API
- Hosting: Render

## Quick Start
```bash
# Local development
go run main.go  # backend on :8080
npm run dev     # frontend on :3000
```
